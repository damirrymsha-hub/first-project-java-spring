First Project Java Spring
Description
This is a simple educational Spring Boot project demonstrating how to build a basic controller, handle HTTP requests, and render an HTML page using Thymeleaf. The project also shows how to pass parameters from the URL into a view and display an image from the static resources folder.

Technologies Used
Java 17
Spring Boot
Maven
Spring Web
Thymeleaf
Lombok
